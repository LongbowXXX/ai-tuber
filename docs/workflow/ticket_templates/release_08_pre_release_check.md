# タスク 8: リリース前確認実施

[← テンプレート一覧に戻る](./README.md)

## チケットタイトル

`[リリース前確認] v{X.Y.Z}`

## チケット内容

```
## リリースバージョン
v{X.Y.Z}

## 確認結果
| カテゴリ | 項目 | 結果 | 備考 |
|----------|------|------|------|
| 動作確認 | {項目1} | ✓/✗ | {備考} |
| 品質 | {項目2} | ✓/✗ | {備考} |
| ドキュメント | {項目3} | ✓/✗ | {備考} |

## 問題・対応
{問題が発見された場合の記録と対応}

## 確認
- [ ] すべての確認項目が実施されている
- [ ] すべての確認項目がパスしている
- [ ] 問題があれば適切に対応されている
```

## 使い方

1. `{X.Y.Z}` を実際のバージョン番号に置き換える
2. タスク 6 で作成した確認項目を実施
3. 結果を表形式で記録
4. 問題があれば対応を記録
5. すべてパスするまで修正を繰り返す
6. 確認結果をチケットに記録
