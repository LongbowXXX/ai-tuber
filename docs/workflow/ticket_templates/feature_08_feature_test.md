# タスク 8: 機能実装完了確認実施

[← テンプレート一覧に戻る](./README.md)

## チケットタイトル

`[機能確認] {機能名}`

## チケット内容

```
## 対象機能
{機能名}

## 確認項目
{タスク5で作成した確認項目を参照}

## 確認結果
| 項目 | 結果 | 備考 |
|------|------|------|
| {項目1} | ✓/✗ | {備考} |
| {項目2} | ✓/✗ | {備考} |

## 問題・対応
{問題が発見された場合の記録と対応}

## 確認
- [ ] すべての確認項目が実施されている
- [ ] 正常系・異常系の両方が確認されている
- [ ] 結果が記録されている

## 成果物
- [ ] 確認結果(プロジェクト管理ツール上に記録)
```

## 使い方

1. `{機能名}` を実際の機能名に置き換える
2. タスク 5 で作成した確認項目を実施
3. 正常系・異常系・境界値すべてを確認
4. 結果を表形式で記録
5. 問題があれば対応を記録
6. 確認結果をプロジェクト管理ツール上に記録

## 依存関係

- **前提**: タスク 5（機能実装完了確認項目作成）、タスク 6（実装・テスト）
- **後続**: タスク 9（ExitCriteria 確認）
