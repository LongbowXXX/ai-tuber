# タスク 9: ExitCriteria 作成

[← テンプレート一覧に戻る](./README.md)

## チケットタイトル

`[ExitCriteria] v{X.Y.Z}`

## チケット内容

```
## リリースバージョン
v{X.Y.Z}

## ExitCriteria項目
### リリース前確認
- [ ] リリース前確認がすべて実施され、パスしている

### リリース実施
- [ ] リリースページが作成されている
- [ ] リリース案内が送信されている
- [ ] Gitタグが作成されている
- [ ] リリースアーティファクトが公開されている

## 成果物
- [ ] ExitCriteria文書(プロジェクト管理ツール上に記録)
```

## 使い方

1. `{X.Y.Z}` を実際のバージョン番号に置き換える
2. リリース前確認の完了を確認項目に追加
3. リリース実施の各項目を確認項目に追加
4. 必要に応じて項目を追加・削除
5. ExitCriteria 文書をプロジェクト管理ツール上に記録
