# タスク 11: ExitCriteria 確認

[← テンプレート一覧に戻る](./README.md)

## チケットタイトル

`[リリース完了確認] v{X.Y.Z}`

## チケット内容

```
## リリースバージョン
v{X.Y.Z}

## ExitCriteria確認結果
- [ ] リリース前確認がすべて実施され、パスしている
- [ ] リリースページが公開されている
- [ ] リリース案内が送信されている
- [ ] すべてのドキュメントが最新の状態に更新されている
- [ ] チームの承認が得られている

## 確認コメント
{各ExitCriteria項目の確認結果詳細}

## 完了判定
- [ ] リリース完了
```

## 使い方

1. `{X.Y.Z}` を実際のバージョン番号に置き換える
2. タスク 9 で作成した ExitCriteria 文書を参照
3. すべての項目が満たされているか確認
4. 確認結果をチケットのコメントに記録
5. すべてクリアしていればリリース完了

## 依存関係

- **前提**: タスク 10（リリース実施）
- **後続**: なし（リリース完了）
