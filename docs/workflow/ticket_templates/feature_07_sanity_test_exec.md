# タスク 7: サニティテスト実施

[← テンプレート一覧に戻る](./README.md)

## チケットタイトル

`[サニティテスト] {機能名}`

## チケット内容

```
## 対象機能
{機能名}

## 確認項目
{タスク4で更新した確認項目を参照}

## 確認結果
| 項目 | 結果 | 備考 |
|------|------|------|
| {項目1} | ✓/✗ | {備考} |
| {項目2} | ✓/✗ | {備考} |

## 問題・対応
{問題が発見された場合の記録と対応}

## 確認
- [ ] すべての確認項目が実施されている
- [ ] 結果が記録されている
- [ ] 問題があれば適切に対応されている

## 成果物
- [ ] 確認結果(プロジェクト管理ツール上に記録)
```

## 使い方

1. `{機能名}` を実際の機能名に置き換える
2. タスク 4 で更新したサニティテスト項目を実施
3. 結果を表形式で記録
4. 問題があれば対応を記録
5. 確認結果をプロジェクト管理ツール上に記録

## 依存関係

- **前提**: タスク 4（サニティテスト項目更新）、タスク 6（実装・テスト）
- **後続**: タスク 9（ExitCriteria 確認）
