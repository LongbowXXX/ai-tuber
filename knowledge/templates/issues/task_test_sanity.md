# サニティテスト結果テンプレート

## 概要

主要機能が正しく動作しているかを確認するための、簡易チェック（ヘルスチェック）結果を記録するテンプレートです。

## 構造

```markdown
# [Test] サニティテスト結果

## 環境

- バージョン: {コミットハッシュ / バージョン}

## チェックリスト参照

Git 上の最新サニティチェックリスト（`docs/tests/sanity.md`）に従って実行してください。

## 実行

- [ ] すべてのチェック項目が実行されている。
- [ ] 結果が記録されている。
- [ ] 発見された課題が対応されている。

**実行結果**

| バージョン    |    結果     | 日付         | 実行者 | 補足                      |
| :------------ | :---------: | :----------- | :----- | :------------------------ |
| {Commit Hash} | PASS / FAIL | {YYYY-MM-DD} | {Name} | All OK / Failed at ST-003 |
```
