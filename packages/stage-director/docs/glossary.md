# 用語集

## ドメイン用語

| 用語                             | 定義                                                           |
| :------------------------------- | :------------------------------------------------------------- |
| **Stage Director**               | 本プロジェクトの名前。AI とレンダリングエンジンの仲介役。      |
| **vtube-stage**                  | キャラクターを表示し、音声やアニメーションを再生するエンジン。 |
| **MCP (Model Context Protocol)** | AI エージェントが外部ツールを利用するための標準プロトコル。    |
| **Speak**                        | キャラクターに発話させるアクション。音声合成と口パクを含む。   |
| **Trigger Animation**            | キャラクターに特定のポーズや動きをさせるアクション。           |

## 技術用語

| 用語                         | 定義                                                               |
| :--------------------------- | :----------------------------------------------------------------- |
| **FastMCP**                  | MCP サーバーを簡単に構築するための Python ライブラリ。             |
| **SSE (Server-Sent Events)** | MCP サーバーが AI エージェントと通信するために使用する方式の一つ。 |
| **UUID**                     | 各コマンドを識別するための一意な ID。                              |
| **Command Queue**            | 送信待ちのコマンドを保持する待ち行列。                             |
| **uv**                       | Python の高速なパッケージ管理・実行ツール。                        |
