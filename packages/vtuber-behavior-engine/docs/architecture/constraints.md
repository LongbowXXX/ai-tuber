# 制約事項

## 技術的制約

- **API レート制限**: Gemini API および Google Cloud Speech API のクォータ制限に注意が必要。
- **レイテンシ**: 音声認識から対話生成、演出実行までの一連のフローには数秒の遅延が発生する可能性がある。
- **コンテキストサイズ**: LLM のコンテキストウィンドウには上限があるため、長期記憶（ChromaDB）からの適切な情報抽出が不可欠。

## プラットフォームの制約

- **Windows/Linux 互換性**: `pyaudio` などのライブラリが OS 依存の挙動を示す場合がある。
- **MCP サーバー依存**: `vtube-stage` が起動していない場合、キャラクターの演出制御ができない。

## 既知の課題

- **感情表現の精度**: LLM が生成する感情タグと、実際のキャラクターモデルの表情が必ずしも一致しない場合がある。
- **音声認識の誤り**: 騒音環境下や滑舌によっては音声認識の精度が低下し、意図しない対話が生成されることがある。
