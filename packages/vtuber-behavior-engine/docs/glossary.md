# 用語集

## ドメイン用語

| 用語                             | 定義                                                                 |
| :------------------------------- | :------------------------------------------------------------------- |
| **ADK (Agent Development Kit)**  | Google が提供する、LLM エージェントを構築するためのフレームワーク。  |
| **MCP (Model Context Protocol)** | AI モデルが外部ツールやデータソースと通信するための標準プロトコル。  |
| **Stage Director**               | キャラクターの演出（発話、アニメーション）を管理する MCP サーバー。  |
| **Character Agent**              | 特定のペルソナ（性格、口調）を持つ対話エージェント。                 |
| **News Agent**                   | ニュース記事を読み込み、それに基づいた解説や議論を行うエージェント。 |
| **Presentation Agent**           | スライド資料に沿ってプレゼンテーションを進行するエージェント。       |

## 技術用語

| 用語                 | 定義                                                                                   |
| :------------------- | :------------------------------------------------------------------------------------- |
| **ChromaDB**         | ベクトルデータベースの一種。会話履歴や知識をベクトル化して保存し、類似検索に使用する。 |
| **Gemini**           | Google の大規模言語モデル (LLM)。本プロジェクトの推論エンジンとして使用。              |
| **uv**               | Rust 製の高速な Python パッケージマネージャー。                                        |
| **Vector Embedding** | テキストを数値のベクトルに変換すること。意味的な類似度を計算するために使用。           |
