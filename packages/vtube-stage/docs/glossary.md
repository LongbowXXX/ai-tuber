# 用語集

## ドメイン用語

| 用語               | 定義                                                                                                |
| :----------------- | :-------------------------------------------------------------------------------------------------- |
| **VRM**            | 3D 人型アバターのためのファイル形式。人型モデルのデータだけでなく、表情や視線などの設定も含まれる。 |
| **BlendShape**     | 表情を作るための仕組み。複数の形状を混ぜ合わせることで、笑う、怒るなどの表情を作る。                |
| **VRMA**           | VRM アニメーション形式。VRM モデルに適用可能なモーションデータ。                                    |
| **リップシンク**   | 音声に合わせてキャラクターの口を動かす技術。                                                        |
| **Stage Director** | V-Tuber システムの司令塔となるバックエンド。AI の思考に基づき、フロントエンドにコマンドを送る。     |

## 技術用語

| 用語           | 定義                                                                             |
| :------------- | :------------------------------------------------------------------------------- |
| **Three.js**   | Web ブラウザ上で 3D グラフィックスを描画するための JavaScript ライブラリ。       |
| **WebSocket**  | サーバーとクライアント間でリアルタイムに双方向通信を行うためのプロトコル。       |
| **TTS**        | Text-to-Speech。テキストを音声に変換する技術。                                   |
| **OBS Studio** | 配信・録画用のオープンソースソフトウェア。本システムの出力を取り込んで配信する。 |
